## 🐛 PROBLEMAS COMUNS

### ❌ Backend não inicia
```bash
# Verificar porta 3001
lsof -i :3001

# Se estiver ocupada, matar processo
kill -9 <PID>

# Reinstalar
npm install
npm start
```

### ❌ Frontend não inicia
```bash
# Limpar cache
rm -rf node_modules
npm install

# Iniciar
npm run dev
```

### ❌ Erro 404 nas APIs
- ✅ Verificar se backend está rodando
- ✅ Verificar URL: `http://localhost:3001`
